Platform Abstraction Types
==========================

Header: ``modula/Common/Include/UrmPlatformAL.h``

Platform-level types, enums, and macros shared between client applications
and the URM server — including ``SysResource``, ``RequestPriority``, ``Modes``,
and all resource/signal code construction macros.
